Portfolio Didah Paridah - Minimal Modern

Cara pakai:
1. Buka file app.js.
2. Ganti YOUR_GITHUB_USERNAME dengan username GitHub kamu.
3. Jalankan index.html di browser.

Untuk menjalankan sebagai web server lokal:
- Buka terminal di folder ini
- Jalankan: python -m http.server 8000
- Buka: http://localhost:8000

File utama:
- index.html
- style.css
- app.js
- app.jss (salinan app.js jika diminta nama file app.jss)
