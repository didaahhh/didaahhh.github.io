# Portfolio Didah Paridah

Portfolio website dengan desain warna-warni soft, modern, dan tetap profesional.

## Isi file

- `index.html`
- `style.css`
- `app.js`
- `app.jss`
- `server.js`
- `package.json`

## Cara menghubungkan GitHub

Buka file `app.js`, lalu ganti:

```js
const githubUsername = "YOUR_GITHUB_USERNAME";
```

menjadi username GitHub kamu, contoh:

```js
const githubUsername = "didahparidah";
```

## Cara menjalankan di VS Code

Buka terminal di folder ini, lalu jalankan:

```bash
node server.js
```

Buka browser:

```text
http://localhost:3000
```

## Upload ke GitHub Pages

Kalau ingin langsung dipakai di GitHub Pages, cukup upload:

- `index.html`
- `style.css`
- `app.js`

File `server.js` dan `package.json` hanya untuk menjalankan server lokal dengan Node.js.
