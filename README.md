# Portfolio Didah Paridah

Desain portfolio warna-warni elegan untuk GitHub Pages.

## File yang dipakai

- `index.html`
- `style.css`
- `app.js`

File `app.jss` disertakan karena sebelumnya diminta, tetapi website memakai `app.js`.

## Cara hubungkan GitHub

Buka `app.js`, lalu ganti:

```js
const githubUsername = "YOUR_GITHUB_USERNAME";
```

menjadi username GitHub kamu, contoh:

```js
const githubUsername = "didahparidah";
```

## Upload ke GitHub Pages

Upload / replace file ini ke repository:

- `index.html`
- `style.css`
- `app.js`

Tidak perlu `server.js`, tidak perlu `package.json`, dan tidak perlu run di VS Code.
